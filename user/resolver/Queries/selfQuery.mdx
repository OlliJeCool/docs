### Arguments

<ParamField body="select" type="FollowingSelect">
Needs to be included if querying the followers field of **User**.
</ParamField>

### Returns

<ParamField query="selfQuery" type="User!">

</ParamField>


<RequestExample>
```graphql Request
query ($select: FollowingSelect) {
  me(select: $select) {
    id
    name
    from
    about 
    verified
    followers {
      followedByCount
      followingCount
    }
  }
}
```
```json Arguements
{
  "select": {
    "followedBy": true,
    "following": true
  }
}
```
```json Headers
{
    "Authoriaztion": "Bearer [jwt]"
}
```
</RequestExample>
